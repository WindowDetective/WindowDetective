Defining Custom Window Data

Various parts of Window Detective can be customised by editing files in the user appdata directory. The following sections list each asset under that directory, what they are used for, and how to modify or add to them.

In Windows XP, the user data directory is: C:\Documents and Settings\<username>\Application Data\Window Detective\
In Vista and Windows 7, it is: C:\Users\<username>\AppData\Roaming\Window Detective\

Window Messages

File: data\window_messages.ini

In addition to the system messages, applications can define their own message to use for communicating with their own windows, or for communicating with other applications. For example; Process Hacker (another great open-source project) defines custom messages for it's graph window class.

If you want Window Detective to recognise such messages, and display their name instead of just the id, then you can add them to window_messages.ini

Window Class Icons

Directory: data\window_class_icons\

Image files under this directory are used by Window Detective as icons for window classes. These icons are shown in the window tree.

You can add images for custom window classes, or to override Window Detective's image for system classes. The name of the file must be the same as the name of the window class you wish to use it for (case-sensitive), and the format must be either PNG or ICO.
