Styles Preferences

You can choose from a number of different styles or "skins" for Window Detective. These change the look and feel of the application's interface.

In addition to these skins, you can also define your own style using Qt style sheets. In the styles directory under the Window Detective user data directory, there are a number of CSS files where you can define styles that apply to a particular part of the application (PropertiesWindow.css, InfoWindow.css), or the whole application (Application.css).

For example, the following code will give all widgets a yellow background and buttons a light red background:
QWidget { background-color: rgb(255, 240, 200); }
QPushButton { background-color: rgb(255, 200, 200); }
